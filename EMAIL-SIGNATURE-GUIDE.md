### Step 2: Replace Placeholders

Update these fields with your information:
- **YOUR NAME** — Your full name (e.g., Sarah Chen)
- **YOUR TITLE** — Your job title (e.g., Senior Account Manager)
- **YOUR.EMAIL@curri.com** — Your Curri email address
- **YOUR PHONE** — Your phone number (optional — remove if not needed)

## Setup by Email Client

### Gmail

1. Open Gmail and click the **gear icon** → **See all settings**
2. Scroll down to the **Signature** section
3. Click **+ Create new** and name it "Curri"
4. Paste the HTML code above
5. Update the placeholders with your info
6. Click **Save Changes** at the bottom

### Outlook (Desktop)

1. Go to **File** → **Options** → **Mail** → **Signatures**
2. Click **New** and name it "Curri"
3. Paste the HTML code in the edit box
4. Update the placeholders with your info
5. Set as default for new messages and replies
6. Click **OK**

### Apple Mail

1. Go to **Mail** → **Settings** → **Signatures**
2. Click **+** to add a new signature
3. Paste the HTML code
4. Update the placeholders with your info
5. Drag the signature to your email account

## Design Elements

### The Route Line

The signature features a distinctive "route line" — a broken horizontal line that subtly evokes Curri's delivery routes:

```
━━━━━━━━━━━━━━━━━━━━  ━━━  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

This is a brand-unique element that differentiates our signatures.

### Color Palette

- **Primary text:** #0a0a0a (near black)
- **Secondary text:** #666666 (dark gray)
- **Links/contact:** #444444 (medium gray)
- **Tagline:** #999999 (light gray)

### Typography

Uses system fonts for maximum compatibility:
- Apple devices: SF Pro
- Windows: Segoe UI
- Android: Roboto
- Fallback: Helvetica Neue, Arial

## Phone Number Guidelines

- **Include your phone** if you regularly take external calls
- **Omit the phone line** if you prefer email-only communication
- Format: (555) 123-4567 (US standard with area code)

## Troubleshooting

### Images not showing?
Some email clients block external images by default. Recipients can click "Display images" to see the logo.

### Signature looks different in replies?
Some clients compress signatures in replies. This is normal behavior.

### Logo appears too large?
Make sure you're using the provided code exactly — don't resize the logo manually.
